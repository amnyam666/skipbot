# Telegram Bot — Учёт пропусков

Бот для учёта пропусков. Позволяет отметить отсутствие и просмотреть историю пропусков.

## Возможности

- Кнопка **"Меня сегодня не будет"** — записывает дату отсутствия в базу данных
- **"Мои пропуски"** — показывает все записанные пропуски пользователя
- Защита от дублей — нельзя отметиться дважды за один день
- Данные хранятся в SQLite (`attendance.db`)

## Установка

1. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

2. Создайте файл `.env` и укажите токен:

```
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

(Пример в файле `.env.example`)

## Запуск

### Локальный запуск

```bash
pip install -r requirements.txt
python bot.py
```

### Запуск в Docker

```bash
# Сборка и запуск
docker compose up -d

# Просмотр логов
docker compose logs -f

# Остановка
docker compose down
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню с кнопками |
| `/absent` | Отметить отсутствие сегодня |
| `/myabsences` | Посмотреть мои пропуски |
| `/help` | Справка по командам |
